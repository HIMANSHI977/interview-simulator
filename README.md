The Interview Simulator is a web-based application designed to help users practice and evaluate their interview skills in a structured environment. It provides a streamlined interface for attempting questions, managing time, and reviewing performance.

🚀 Features
Structured question flow
Timer-based interaction
Real-time score tracking
Result summary at the end of the session
Start, pause, and restart functionality
Clean and responsive user interface
🛠️ Tech Stack
HTML
CSS
JavaScript
⚙️ Functionality

The application initiates an interview session where users are presented with questions sequentially. A timer controls the duration of the session, encouraging time-bound responses. As the user progresses, their inputs are processed and evaluated.

At the end of the session, the system displays a summary of the user’s performance, including their score. The user can pause the session or restart it at any time, allowing for repeated practice and improvement.
